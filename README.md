# Music Recommendation System

A machine learning-based **Music Recommendation System** developed using **Python** and **MongoDB** to provide personalized song recommendations based on song similarity. The system uses **content-based filtering** to suggest songs with similar features, helping users discover music that matches their preferences.

## Features

- Personalized song recommendations
- Content-based filtering algorithm
- Song similarity analysis
- MongoDB database integration
- Efficient data preprocessing
- User-friendly recommendation system

## Technologies Used

- Python
- MongoDB
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/music-recommendation-system.git
```

### Navigate to the project directory

```bash
cd music-recommendation-system
```

### Install the dependencies

```bash
pip install -r requirements.txt
```

## Running the Project

```bash
python app.py
```

## How It Works

1. Load the music dataset.
2. Preprocess the song metadata.
3. Generate feature vectors for each song.
4. Calculate similarity using content-based filtering.
5. Recommend songs similar to the selected track.
